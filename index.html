Embark on an unforgettable journey and battle against formidable enemies.

1. Click the download button
2. Right click on zip file and select the "Extract all" option
3. Double click the file named "platformer.exe" with file type "Application"
If you would like to create a desktop shortcut, click once on the file, right click on the file, select "Show more Options," and then select "Create shortcut." Then drag the shortcut file to your desktop.

How to play:

Use the level selector to select your desired level. Using the arrow keys and the number row, attempt to slay the boss before you lose all your health. Make sure to watch out for their minions before its too late!
